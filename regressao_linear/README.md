## 📈 Data Science: Testando Relações com Regressão Linear
Este repositório contém os notebooks desenvolvidos durante o curso "Data Science: testando relações com Regressão Linear" da Alura. O foco do curso é compreender e aplicar modelos de regressão linear para analisar e prever preços de casas a partir de suas características.

## 📚 Conteúdo do Curso
- Ajustando uma reta
- Explicando a reta
- Adicionando outros fatores
- Precificando as casas
- Investigando o modelo

## 📁 Notebooks
Preços_de_Imóveis_Utilizando_Regressão_Linear.ipynb

## 🔧 Tecnologias Utilizadas
- Python 3.x
- Pandas
- NumPy
- Scikit-learn
- Statsmodels
- Matplotlib / Pyplot
- Seaborn (incluindo pairplot)
- Google Colab

## 🏡 Problema Trabalhado
O desafio do curso foi prever preços de casas com base em suas características, explorando como diferentes variáveis impactam o valor final e como avaliar a qualidade de um modelo de regressão linear.
